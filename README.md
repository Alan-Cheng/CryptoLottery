# lottery_project

## 介紹
- 利用python flask + SQLite +ngrok簡單架設一個Linebot
- 可以提供類似樂透的功能

## 需求
- 開發者需事先準備line developer的帳號，取得token與key才能使用
- 需自行在環境中使用ngrok讓外網可連接到API

## 功能
- 顯示比特幣價格
- 下單購買樂透（目前只有隨機選號）
- 顯示擁有的彩券
- 顯示當期樂透號碼（顯示當期台彩彩券號碼，因故未實作)
